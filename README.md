# KeepTabs

Save your browser tabs into named folders, close them, and reopen them later —
without losing anything. **No account, no servers, no tracking. Your data stays
on your device.**

## Why

Tab managers exist, but many of them lose your saved tabs, force you to create an
account, or hide core features behind a paywall. KeepTabs is the opposite:

- 🗂️ **Save tabs into named folders** ("work", "research", "youtube"…)
- ↩️ **Reopen** a whole folder or a single tab anytime
- 💾 **One-click backup** — export everything to a JSON file
- 🔒 **Local only** — nothing leaves your computer, no account, no server
- 🌗 **Light & dark** — follows your system theme

## Privacy

KeepTabs collects nothing. All data is stored locally via `chrome.storage.local`.
See [privacy.html](privacy.html).

## Tech

Plain HTML/CSS/JavaScript. Manifest V3. Permissions: `tabs`, `storage`.
